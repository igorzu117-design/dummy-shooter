# Dummy Shooter

A fast-paced 3D shooter built with Three.js. Fight against bots, capture flags, and upgrade your gear in the shop!

## 🎮 How to Play
- **WASD**: Move
- **Mouse**: Aim & Shoot
- **Space**: Jump
- **R**: Reload
- **B**: Open Shop
- **1-5**: Select Weapon
